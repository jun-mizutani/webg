webg unittest

unittest は、公開 sample 一覧とは分けて管理する内部確認用の置き場です。

この一覧の目的

ここでは、単体確認、回帰確認、最小表示確認に向いた app をまとめます。

外部へ見せる sample は samples/index.html に残し、ここではテスト対象 1 件だけを確認しやすい構成を優先します。

文字表示と情報表示の整理は 14_UI表示の設計.md にまとめています。`message` は短い ASCII HUD、`dialogue` は UTF-8 の会話を scene の上へ重ねる DOM overlay、`theme` は `DebugDock` / `FixedFormatPanel` / `UIPanel` の比較として読むと役割が見えやすくなります。

`message`、`touch`、`tilemap` は、raw `Message` や最小 HUD を残して挙動を切り分ける内部 test です。新しい sample の標準経路としては `breakout`、`bloom` / `dof`、`scene`、`dialogue` を先に参照するほうが判断しやすくなります。

今後は debug mode 固定、UI 最小化、共通 helper 化をさらに進める予定です。現在は `background`、`message`、`cube_axes`、`touch`、`raycast` が `_shared/UnitTestApp.js` を使う移行例で、`theme` は `WebgApp` の UI theme 切替確認用 app です。`axis` は `samples/axis` に移し、座標系と FOV を学ぶ教育用 sample として扱います。`phong_debug` は `Phong`、`NormPhong`、`BonePhong`、`BoneNormPhong` の debug を見比べる test で、折れ板と筒状メッシュで表裏や normal map、weight debug を確認できます。`game_api` は game 向けに追加した action map、scene state、collision body、HUD helper をまとめて見る test です。`input_controller` は `InputController` の physical key / action map / pulse action を自動確認したあと、touch ボタンも含めて手で確認する test です。`tween` は `Tween`、`Node.animateRotation()`、`Shape.animateParameter()`、`Space.updateNodeAnimations()`、`Space.updateShapeAnimations()`、`WebgApp.createTween()`、`shakeCamera()`、`flashMessage()` の動きをまとめて確認する test です。`particle_emitter` は `ParticleEmitter` の preset / emit / update / draw / clear を自動確認したあと、WebgApp の実画面で burst と preset 切替も確認する test です。`camera_follow` は `WebgApp` の followNode / lockOn / clearCameraTarget を自動確認したあと、canvas 上で target が動き続け、`yaw-follow` では target heading と camera yaw の関係も確認できる test です。`dialogue` は `UIPanel` 上の DOM overlay を使う会話 / tutorial の標準 test で、`Dialogue.start()` / `next()` / `choose()` の考え方を確認します。`tilemap` は core 側へ組み込んだ TileMap の確認用 unittest で、displayArea の scroll と camera pivot の追従も含めて、同じ高さ付き盤面の見え方と移動ルールを build した runtime から確認します。こちらも touch で orbit / pan / zoom を確認できます。旧 `tilemap_poc` と `tile_shape_poc` は ../../discontinued/index.html に保管しました。`unittest/*/*.txt` には、それぞれの test が何を確認しているかを文章で残しています。

詳細テキスト

  • background.txt: 背景画像の全画面描画、色乗算、アスペクト補正を確認する説明
  • cube_axes.txt: `addVertex` / `addPlane` のみで作る立方体と軸回転を確認する説明
  • detouch_min.txt: `attach` / `detach` で world 位置を保つ確認をまとめた説明
  • game_api.txt: ゲーム向け action map、scene state、collision body、HUD helper の確認説明
  • input_controller.txt: `InputController` の action map、pulse action、touch/manual 確認の説明
  • tween.txt: `Tween`、`Node.animateRotation()`、`Shape.animateParameter()`、`shakeCamera()`、`flashMessage()` の確認説明
  • particle_emitter.txt: `ParticleEmitter` の auto test と visual sample の確認説明
  • camera_follow.txt: `WebgApp` の camera follow と実画面 visual sample の確認説明
  • dialogue.txt: `UIPanel` 上の DOM overlay で会話と tutorial の順番表示を確認する説明
  • embedded.txt: `WebgApp` の embedded layout で canvas と overlay が本文スクロールに追従するかを確認する説明
  • tilemap.txt: core に組み込んだ TileMap の runtime 化と表示域 scroll / 移動確認説明
  • smooth_shader.txt: `SmoothShader` 1 本で 4 経路をまとめて確認する説明
  • phong_debug.txt: `Phong` 系シェーダの裏面表示、normal map、weight / UV の確認説明

visual test

  • cube_axes: addVertex / addPlane だけで組んだ手組み mesh の基本確認
  • detouch_min: attach / detach の world 位置保持だけを確認する最小 test
  • smooth_shader: SmoothShader 1 本で static / skinned / normal map の 4 経路を確認する test
  • skinning_basic: SmoothShader の最小 skinning 経路を確認する test
  • skinning_normal_map: SmoothShader の normal map 付き skinning 経路を確認する test
  • phong_debug: Phong、NormPhong、BonePhong、BoneNormPhong の debug をまとめて確認する test
  • vignette: VignettePass の offscreen -> fullscreen 合成経路を確認する最小 test
  • primitive_texture_uv: primitive ごとの texture mapping と UV 崩れを最小構成で確認する test
  • primitive_wireframe: primitive 群の solid / wireframe 切替を最小構成で確認する test
  • primitive_normal_map: SmoothShader で primitive と normal source の違いを 3x3 で見比べる最小 test
  • primitive_modelasset: Primitive から ModelAsset / validator / builder 経路を最小確認する test
  • game_api: action map、GameStateManager、collision helper、HUD helper をまとめて確認する test
  • input_controller: `InputController` の key normalize、action pulse、touch/manual 確認を行う test
  • tween: `Tween`、`Node.animateRotation()`、`Shape.animateParameter()`、`shakeCamera()`、`flashMessage()` を確認する test
  • particle_emitter: `ParticleEmitter` の auto test 後に実画面で burst と preset 切替を確認する test
  • camera_follow: `WebgApp` の followNode / lockOn / clearCameraTarget を自動確認し、canvas 上で角のある target 追従と yaw-follow の向き継承を確認する test
  • dialogue: `UIPanel` 上の DOM overlay で会話と tutorial の順番表示と branch 差し込みを確認する test
  • embedded: `WebgApp` の `layoutMode: "embedded"` と `fixedCanvasSize` を組み合わせ、本文中に置いた canvas と HelpPanel / dialogue / fixed panel / touch が scroll 中も同じ host を基準に動くかを確認する test
  • tilemap: core に組み込んだ TileMap の runtime 化、orbit camera、displayArea scroll と camera pivot 追従、camera 向きに合わせた矢印キー移動、click / tap 選択、touch の orbit / pan / zoom を確認する test
  • theme: `WebgUiTheme` の preset 切替で debug dock / `FixedFormatPanel` / `UIPanel` の見え方を確認する test
  • touch: `Touch.js` の hold / action 入力と PC 常時表示の touch ボタンを確認する test
  • raycast: マウス座標からの ray 生成と `space.raycast` のヒット判定を確認する test
