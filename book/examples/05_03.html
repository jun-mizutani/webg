05_03 fixedCanvasSize と embedded

`book/05_WebgAppによるアプリ構成.md` の `fixedCanvasSize`、`layoutMode: "embedded"`、`EyeRig`、`takeScreenshot()` を 1 ページでまとめて試せる埋め込み例です

このページは本文スクロールを残したまま canvas と help panel を一緒に埋め込む形にしてあります。マウスドラッグでカメラを回し、`S` でスクリーンショット保存を試してください

前置きの本文です。教材ページでは、3D の説明文を読み進めながら、その途中で実際の描画結果を確認したくなることがあります

この例では `layoutMode: "embedded"` を使って、canvas と help panel が本文の流れの中に入るようにしています。ページ全体は通常どおりスクロールでき、canvas の周囲に説明文やコードを置けます

canvas の下にも説明文を続けられます。`fixedCanvasSize` を使うと、比較用スクリーンショットや教材のレイアウト調整で、表示サイズを一定に保ちやすくなります

この例は添付してもらった `user/WebgApp01.html` の構成を整理し直して、`book/examples` 用の相対 path へ合わせたものです
