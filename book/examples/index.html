book/examples

`book` 各章の実行用 HTML を、ファイル名と確認目的つきで一覧にした index です

各ページは本文コードを 1 ファイルで試すための確認用ページです。書籍本文を読みながら、描画結果や UI の見え方を対応づけるときに使えます

各サンプルはウィンドウのフォーカスが外れると、省電力のためアニメーションが停止します。
