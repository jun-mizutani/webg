webg sample: webgmodeler

`ModelAsset` を編集データとして扱う初期版の 3D ポリゴンモデラーです。 canvas は 3D ビューポート、読み込み・保存・編集コマンドは DOM UI で操作します。

左ドラッグで orbit、Shift + 左ドラッグで PAN、wheel で zoom。 クリックで選択、Shift + クリックで選択の追加/解除です。編集移動は J/L/I/K、U/O、N/M を使います。

ファイル

GLB / Collada / ModelAsset JSON を読み込み、編集対象 mesh を選択できます。

編集

Triangle / Quad は選択頂点数が 3 / 4 のときに作成します。 Extrude は選択 face を法線方向へ押し出します。選択頂点の移動や scale はキー操作で行います。
