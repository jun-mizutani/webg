webg sample: webgmodeler

`ModelAsset` を編集データとして扱う初期版の 3D ポリゴンモデラーです。 canvas は 3D ビューポート、読み込み・保存・編集コマンドは DOM UI で操作します。

中ボタンドラッグまたは Option + 左ドラッグで orbit、Shift 併用で PAN、Ctrl 併用または wheel で zoom。 Object Mode でオブジェクト選択、Tab / Edit Mode でアクティブオブジェクトを編集します。Vertex/Face Select でクリック選択、modifier なしの左ドラッグで矩形選択、A で全選択です。 W で Object Mode の wireframe を切り替え、X で現在 mode / tool の選択対象を削除します。G/R/S/E の後はマウス移動で移動/回転/scale/押し出し、左クリックで確定します。F は選択 3 / 4 頂点から面を作成し、補助編集は J/L/I/K、U/O、N/M を使います。

ファイル

ロード後は選択中の mesh が自動で取り込まれます。複数 mesh のファイルでは、select で別 mesh や all objects を選び、Use Mesh で再取り込みできます。

編集

Face は選択頂点数が 3 / 4 のときに三角形 / 四角形を作成します。 移動 / 回転 / scale / 押し出し / 削除は Blender 相当の G / R / S / E / X キーを使います。Flip Face は選択 face の表裏を反転します。
